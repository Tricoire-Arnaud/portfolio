Application de Budgétisation

Description :

URL du Projet: atbudget.vercel.app

L'Application de Budgétisation est un projet web interactif conçu pour aider les utilisateurs à gérer efficacement leurs finances personnelles. Développé en utilisant HTML, CSS et JavaScript, avec l'intégration de bibliothèques telles que Bootstrap et Chart.js, cette application offre une expérience utilisateur intuitive et moderne.

Fonctionnalités clés :

  • Ajout de Transactions : Permet aux utilisateurs d'ajouter rapidement des transactions en spécifiant le type (revenu/dépense), la description, le montant et la catégorie.
  • Liste Dynamique des Transactions : Affiche une liste dynamique des transactions, permettant aux utilisateurs de visualiser leurs opérations financières récentes.
  • Graphique des Transactions : Présente visuellement le solde du budget à travers un graphique interactif, offrant une compréhension instantanée de la santé financière.
  • Filtrage par Catégorie : Permet aux utilisateurs de filtrer les transactions par catégorie, offrant une vue spécifique de leurs dépenses et revenus.
  • Gestion du Solde : Calcule automatiquement le solde du budget en fonction des transactions, fournissant une indication visuelle de la situation financière.

Technologies Utilisées :

  • HTML5, CSS3
  • JavaScript (ES6+)
  • Bootstrap 5
  • Chart.js

Résultats :

Cette application de budgétisation est une démonstration de mes compétences en développement web, mettant en œuvre des fonctionnalités cruciales pour une gestion financière efficace. La conception réactive, l'interface utilisateur conviviale et la visualisation graphique des données en font un outil puissant pour ceux qui cherchent à mieux contrôler leurs finances.
