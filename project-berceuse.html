Contexte du Projet

La Petite Berceuse est un projet de site vitrine pour une infirmière puéricultrice. L'objectif était de créer une présence en ligne professionnelle et chaleureuse, permettant aux jeunes parents de découvrir ses services et de prendre contact facilement.

Technologies

HTML5 CSS3 JavaScript Bootstrap

Stack technique

HTML5 pour la structure
CSS3 pour le style et les animations
JavaScript pour l'interactivité
Bootstrap pour le design responsive

Approche & Valeurs

Le développement de ce site a été guidé par la volonté de créer une expérience rassurante et professionnelle pour les jeunes parents.
